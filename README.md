# Projektübersicht

Dieses Repository enthält eine React/Vite-Anwendung auf Basis von TypeScript, Tailwind CSS und shadcn-ui.

## Lokale Entwicklung

Voraussetzung: Node.js und npm.

```sh
# Repository klonen
git clone <REPO_URL>
cd <PROJEKTORDNER>

# Abhängigkeiten installieren
npm install

# Entwicklungsserver starten (Standard: http://localhost:5173)
npm run dev
```

## Build & Preview

```sh
npm run build   # Produktionsbuild erzeugen
npm run preview # Lokale Vorschau des Builds
```

## Technologien

- Vite
- React mit TypeScript
- Tailwind CSS
- shadcn-ui
